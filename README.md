
# Pokemon Commerce

Web application at Next.js

## Tech Stack

**Client:** Next.JS, React, Redux, Material-UI , Axios, SWR.


## Run Locally

Clone the project

```bash
  git clone https://github.com/allanmonth/pokemon_commerce
```

Go to the project directory

```bash
  cd pokemon_commerce
```

Install dependencies

```bash
  yarn i
```

Start the server

```bash
  yarn dev
```
